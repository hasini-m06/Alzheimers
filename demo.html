How It Works
CogTraceEnv is an OpenEnv-compliant RL training environment
🧠
Synthetic Patient
Rule-based simulator generates realistic behavioral signals derived from published CDR-scale research. No real patient data used.
📊
5 Sensor Streams
Typing delay, sleep hours, routine adherence, speech pauses, and memory lapses — each with stage-calibrated noise.
⚡
Dense Rewards
Agent receives a reward every step. Missing a critical event costs −1.0. Correct restraint earns +0.10. Proportionate alerting encouraged.
🎯
3 Graded Tasks
Easy: stage classification. Medium: anomaly timing detection. Hard: full 30-step triage with F1 scoring.
